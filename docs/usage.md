# Usage
you use it as a regular calculator

## Basic Operators

- `+` for the addition
- `-` for the subtraction
- `/` for the division
- `*` for the multiplication
- `^` for the exponentiation

## Setting variable

It's easy to set a variable, just use 
``` 
myvar = value
```
example:
```
myvar = (2+2)
```

## Built-in variable

- `ans` is the last calculation done
- `pi` is pi 
- `e` is e
